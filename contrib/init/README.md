Sample configuration files for:
```
SystemD: shitecoind.service
Upstart: shitecoind.conf
OpenRC:  shitecoind.openrc
         shitecoind.openrcconf
CentOS:  shitecoind.init
macOS:   org.shitecoin.shitecoind.plist
```
